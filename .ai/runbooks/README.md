# Runbooks (`/.ai/runbooks`)

**Agents:** use the **[routing hub — `HUB.md`](./HUB.md)** (symptom → one target file). Avoid enumerating this whole directory.

**Maintainers:** deep runbooks stay as leaf `.md` files here; human strategy lives under `docs/maintainers/` only when generated or explicitly maintained there.

**From cold start:** `.ai/WORKSPACE-KIT-SESSION.md` → this README → **`HUB.md`** → chosen runbook (≤3 hops).
