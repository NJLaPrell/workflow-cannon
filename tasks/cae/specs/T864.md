Invoke **shadow** evaluation at designed hook points (task + command merge) so every relevant `wk run` surfaces **trace id** / summary without blocking.

## Scope

**In:** `module-command-router` and/or CLI wrapper changes, feature flag, caching per T849, structured logging or result side channel.  
**Out:** Advisory payload (T865), enforcement (T866).

## Deliverable artifacts

1. **Code changes** with clear single orchestrator call site (document path in PR).
2. **E2E test**: representative command produces shadow metadata (assert on stderr JSON or response extension per design).
3. **Performance note** in PR: measured overhead or cache hit rate smoke test.

## Decisions to resolve

- Commands excluded from hook (e.g. `doctor`, `list-tasks`).
