## What's New in 1.0.10

This release focuses on **Dashboard UX polish** after Phase 148.

### Highlights

- **New Idea** opens in the shared drawer instead of an always-visible form.
- Startup no longer flips a healthy dashboard back to “still loading.”
- Agent Activity stops showing a fake active orchestrator when nothing is running.
- Plans list shows `ID - Title` with cleaner expanded cards; Ideas rows match Wishlist chrome.
- Visual pass: callouts, focus rings, and CSS-variable color/font consistency across dashboard surfaces.

### Also included

- Phase 109 skill-pack backlog drained and retargeted to unphased.
- Task-state snapshot pointer repair for safer hydrate.
