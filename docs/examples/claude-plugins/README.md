# Example Claude-layout plugins

This directory holds **sample plugin directories** for documentation and CI. Each child directory is one plugin root: **`<id>/.claude-plugin/plugin.json`**, matching Anthropic’s documented layout (see **`docs/maintainers/adrs/ADR-claude-code-plugin-platform-v1.md`**).
