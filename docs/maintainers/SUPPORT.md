# Support

## Getting help

- **Usage questions** — open a GitHub issue with clear reproduction details.
- **Feature ideas** — open a GitHub issue and include context plus proposed task direction.
- **Security issues** — use the private reporting path described in `docs/maintainers/SECURITY.md`.

## What to include

- Version or commit SHA of `@workflow-cannon/workspace-kit`.
- OS and Node.js runtime version.
- Exact command(s) run.
- Expected result vs actual result.
- Relevant logs, stack traces, or screenshots.

## Response expectations

- Initial triage target: within 5 business days.
- Priority is based on severity, reproducibility, and impact.
- These are targets, not guarantees — we aim for responsive and transparent communication.
