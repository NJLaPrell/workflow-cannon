# FAQ

## What is Workflow Cannon?

Workflow Cannon is the canonical home for the `@workflow-cannon/workspace-kit` package and related workflow planning/tasking improvements.

## Where should I start?

Read `README.md`, then `.ai/PRINCIPLES.md`, `docs/maintainers/ROADMAP.md`, and `docs/maintainers/TASKS.md`.

## Where do feature ideas go?

Open a feature request issue and link it to `docs/maintainers/ROADMAP.md` / `docs/maintainers/TASKS.md` context.

## How do releases work?

See `docs/maintainers/RELEASING.md` and `docs/maintainers/CHANGELOG.md`.

## How do I report security issues?

Use the private reporting process in `docs/maintainers/SECURITY.md`, not public issues.
